# Mautic-Newsman - Email Integration
[Newsman](https://www.newsmanapp.com) integration for Mautic 2.X. Send Email via newsman smtp api.

# Installation

Manual installation:
1.  Copy contents of the `app` folder and paste to your mautic root directory, create backup of `app` folder first.
2.  Go to Configuration->Email Settings->Service to send mail through->Newsman

# Setup

1. Fill in your Newsman SMTP API KEY and User ID and click Test Connection & Send test email
